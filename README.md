# Kanna Training Repository

Learning GitHub while building toward Project Kanna.

## What is Kanna?

Kanna is a conversational commerce platform helping local Indian sellers get discovered through AI-powered conversations.

## Tech Stack

- **Frontend:** Next.js + React (Vercel)
- **Backend:** n8n workflows (DigitalOcean)
- **Database:** Supabase (PostgreSQL)
- **AI:** GPT-4 + Gemini Flash

## Folder Structure

- `apps/pwa/` - Next.js PWA code
- `database/` - Migrations and functions
- `n8n/workflows/` - Exported n8n workflows
- `docs/modules/` - Build specifications
- `.github/workflows/` - CI/CD automation
